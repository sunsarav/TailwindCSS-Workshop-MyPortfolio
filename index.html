About Me

Java Full-Stack Developer · Västerås, SE

I'm a full-stack developer with an MSc in Information Technology, currently completing an intensive Java developer training programme at Lexicon, Västerås. My focus is on building clean, well-tested backend systems with Java and Spring Boot, and intuitive frontends with React.
